# VEMS360 - Virtual Event Management System

![VEMS360 Screenshot](./assets/vems360-screenshot.png)
*VEMS360 Dashboard Preview*

**Description:**
VEMS360 is a **web-based event management platform** designed for **virtual, hybrid, and physical events**. It provides a seamless experience for **admins, organizers, and attendees** to manage, track, and participate in events efficiently.

**⚠ Note:** This platform is **optimized for laptop/desktop use only**. Mobile responsiveness is not fully supported.

## Features

* **User Authentication:** Secure registration and login
* **Event Management:** Create, edit, and manage events
* **Feedback & Reporting:** Collect and analyze attendee feedback
* **Email Notifications:** Automated emails via PHPMailer
* **Secure Backend:** PHP + MySQL for data storage and operations

## Tech Stack

* **Backend:** PHP 8, MySQL
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5

## Folder Structure

* `/config` – Database & configuration files
* `/assets` – Images, CSS, JS
* `/admin` – Admin panel functionalities
* `/organizer` – Organizer panel functionalities
* `/events` – Event listing and management

## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/VEMS360.git
   ```
2. Import `vems360.sql` into your MySQL database.
3. Update database credentials in `config/connection.php`.
4. Run the project using XAMPP:

   ```
   http://localhost/VEMS360/
   ```
