# VEMS360 - Virtual Event Management System

**Description:**  
VEMS360 is a web-based event management platform designed for **virtual, hybrid, and physical events**.  
It allows admins, organizers, and customers to manage and participate in events efficiently.

## Features
- User authentication (register/login)
- Event creation & management
- Feedback & reporting system
- Email notifications (PHPMailer)
- Secure PHP + MySQL backend

## Tech Stack
- PHP 8
- MySQL
- HTML5, CSS3, JavaScript
- Bootstrap 5

## Folder Structure
See `/config`, `/assets`, `/admin`, `/organizer`, `/events` for modular code organization.

## How to Run
1. Clone the repo
2. Import `vems360.sql` into your database
3. Update database credentials in `config/connection.php`
4. Run project using XAMPP (htdocs/VEMS360/)
